# ShowTime – ES6 Movie & TV Shows Catalog

**ShowTime** is a multi-page Movie & TV Shows Catalog web application built using pure HTML, CSS, and Vanilla JavaScript (ES6).  
It allows users to register, log in, explore movies and TV shows, filter content by category, rating, and genre, and manage a personal watchlist.  

This project demonstrates modern **ES6 concepts** such as modules, arrow functions, destructuring, spread/rest operators, template literals, array methods, and localStorage for data persistence — all implemented without any external libraries or frameworks.

---

## 🖥 Features

- **User Authentication**: Register & Login with form validation and data stored in `localStorage`.  
- **Home Page**:  
  - Featured movies/TV shows image slider with autoplay, navigation arrows, and dot indicators.  
  - Movie cards displaying poster, title, rating, release year, and add-to-watchlist icon.  
  - Filter movies/TV shows by category, genre, rating, and release year.  
- **Movie Details Page**: View full details of a selected movie/TV show including overview, rating, genre, and release date.  
- **Watchlist Page**: Add, remove, and manage favorite movies/TV shows.  
- **Watched Confirmation Page**: Mark movies/TV shows as watched with confirmation message.  
- **UI Enhancements**:  
  - Hover effects on cards  
  - Scroll to top button  
  - Responsive design (desktop and tablet friendly)

---

## 🎯 Technologies Used

- **HTML5**  
- **CSS3** (Flexbox, Grid, Transitions)  
- **Vanilla JavaScript (ES6)**  
  - `let` / `const`  
  - Arrow functions  
  - Destructuring  
  - Spread / Rest operators  
  - Template literals  
  - Array methods: `map()`, `filter()`, `forEach()`, `find()`, `reduce()`, etc.  
  - Object methods: `Object.keys()`, `Object.values()`, `Object.entries()`, `Object.assign()`  
  - `for...of` and `for...in` loops  
- **localStorage** for user authentication and watchlist persistence
